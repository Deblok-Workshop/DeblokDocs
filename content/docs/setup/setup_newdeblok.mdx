This guide will show you how to deploy NewDeblok on your computer:
<Steps>
  <Step>
    Start by cloning the NewDeblok GitHub repository:
    ```bash
    git clone --submodules https://github.com/Deblok-Workshop/NewDeblok.git
    ```
  </Step>
  <Step>
    Then install dependencies
    ```bash
    npm i -g bun # if you haven't installed bun
    bun i
    ```
  </Step>
  <Step>
    Now add your DeblokManager endpoints to your `.env` file, for example:
    ```
    DBPWD=unused-but-required
    ENDPOINTS=example.com:8000,example2.com:8000
    ```
  </Step>
  <Step>
    Finally start NewDeblok
```
$ bun run start
$ bash scripts/unattended.sh
[i] Building styles...
$ tailwindcss -i static/tw.css -o static/app.css -m
Rebuilding...
Done in 1576ms.
[i] Starting server...
Listening on port 8080 or
 │ 0.0.0.0:8080
 │ 127.0.0.1:8080
 └─────────────────────────>
 ```
  </Step>
</Steps>
